A) Latar Belakang
: Sebelum adanya Node JS. Kode javascript hanya dapat 
dijalankan di browser. Hal ini membuat pengembangan aplikasi
fullstack(FE + BE) harus mempelajari bahasa pemrograman yang berbeda.

B) Definis Node JS
: Node JS ialah javascript runtime yang dapat menjalankan javascript
di luar browser.

C) Menjalankan Javascript menggunakan Node JS
    C.1) Node.js REPL
    : REPL(Read-Eval Print Loop) ialah fitur node 
    untuk membaca, mengevaluasi, dan mencetak ahsil evaluasi
    ke console.

        C.1.1) Command REPL
         C.1.1) node
         : Menjalankan REPL
         C.1.2) .editor
         : Mode editor sehingga tidak perlu perbaris
         C.1.3) ctrl + c
         : Keluar
         C.1.4) ctrl + d
         : Mengeksekusi Kode

    C.2) Dengan File JS
    : Buat file dengan ekstensi .js kemudian pada console ketikan
    node "<nama file>".js
